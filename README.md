# Проект Никиты — Калькулятор

Школьный проект: веб-калькулятор на HTML, CSS и JavaScript.

## Возможности

- Базовые арифметические операции: сложение, вычитание, умножение, деление
- Поддержка процентов (`%`)
- Ввод с клавиатуры
- Удаление последнего символа (⌫)
- Полный сброс (AC)
- Красивый тёмный интерфейс

## Запуск

Откройте файл `index.html` в любом современном браузере.

## Клавиши

| Клавиша | Действие |
|---------|----------|
| `0`–`9`, `.` | Ввод цифр |
| `+`, `-`, `*`, `/`, `%` | Операторы |
| `Enter` или `=` | Вычислить |
| `Backspace` | Удалить последний символ |
| `Escape` | Очистить всё |

## Структура проекта

```
projectNikita/
├── index.html   # Основная HTML-страница
├── style.css    # Стили (тёмная тема)
└── script.js    # Логика калькулятора
```
